**Training Requirements**

**Prerequisites**

Coordinator confirms each trainee has recorded prior competencies (on file in QMS/SharePoint) for: 

    - Basic NGS concepts (FASTQ, reference genomes, mapping, variant calling). 
    - Intro Galaxy use (histories, datasets, tools, running jobs). 

**Evidence required before enrolment:** prior certificates/training records or brief competence check (short task or quiz). 

**If not met:** trainee completes prerequisite training before joining this module.
