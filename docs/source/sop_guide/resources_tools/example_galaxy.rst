**Resources and Tools**

    - Computer or laptop with stable internet connection. 
    - Access to the Galaxy server (usegalaxy.org or institutional Galaxy instance). 
    - TB dataset (FASTQ files from Illumina sequencing). 
    - Training slides and tutorial handouts (PDF / SharePoint). 
    - Attendance and feedback forms.
