**Purpose**

This SOP defines the laboratory’s standardised procedure for the preparation of sequencing libraries using the Illumina DNA Prep Library Preparation Kit. The procedure supports the laboratory’s sequencing workflows by enabling the reliable conversion of extracted DNA into dual-indexed libraries.
