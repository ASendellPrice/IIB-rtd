**Purpose**

The purpose of this SOP is to provide standardized guidance for training users in the development, execution, and management of analysis workflows within the Galaxy platform. This document ensures that trainees gain the knowledge and practical skills needed to navigate the Galaxy environment, create reproducible computational workflows, run analyses accurately, troubleshoot common issues in Galaxy, and maintain proper documentation within Galaxy. Establishing a consistent training procedure promotes consistent skill development and ensures competency assessment.
